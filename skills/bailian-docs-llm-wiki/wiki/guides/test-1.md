# test 1

本文档汇总了阿里云百炼平台的大模型计费规则、额度管理与成本优化方案，涵盖模型推理、训练、部署及账单分摊等核心环节。开发者可参考本指南快速掌握价格体系、节省计划配置及账单查询方法，以实现精细化成本控制与服务集成。

## 支持的模型与功能
- **实时推理与高级特性**：覆盖文本生成、视觉与语音系列模型。完整支持 [[batch-inference]]（半价计费）与 [[context-cache]]（输入Token折扣），两者互斥生效。
- **训练与定制**：支持千问（文本/VL）与万相（视频）系列的 [[model-training]]，按训练Token总量与循环次数计费。微调后需完成 [[model-deployment]] 方可推理调用。
- **部署模式**：提供预置吞吐（TPM）、模型单元（MU）时长、实例独占及自定义模型按Token量计费四种模式，满足不同并发与延迟需求。
- **成本优化方案**：提供 AI 通用型节省计划、垂直模型节省计划及资源包。详细抵扣范围与生效规则请参考 [新人免费额度](../../raw/model-user-guide/test-1/new-free-quota.md) 与 [节省计划与资源包](../../raw/model-user-guide/test-1/savings-plan-and-resource-package.md)。

## 关键参数
- **计费粒度**：推理按输入/输出 Token 计费（最小单位 1 Token）；训练按 `训练Token总量 × 循环次数 × 单价` 计算；部署按资源占用时长或购买的 TPM/MU 规格结算。
- **阶梯阈值**：部分文本模型按单次请求输入 Token 划分区间（如 0~32K、32K~128K、>128K），单次请求全量 Token 均按命中区间的单价结算。
- **额度周期**：免费额度有效期为 30～90 天，自开通或申请通过起算；节省计划按月承诺消费，当月额度未使用完自动清零，不累积至下期。
- **出账延迟**：大模型推理账单通常在调用后 2～10 分钟生成（分钟级）；批量推理、训练与知识库等服务为小时级出账。

## 使用方式
- **额度管理与防超耗**：开通平台后自动发放中国内地版实时推理额度。建议在控制台“模型用量”页开启 `免费额度用完即停`，额度耗尽时 API 将返回 `AllocationQuota.FreeTierOnly` 并拦截计费。
- **购买与抵扣顺序**：资源包与节省计划购买后立即生效，无需手动激活。系统严格遵循抵扣链路：**免费额度 → 资源包 → 其他模型节省计划 → AI 通用型节省计划 → 按量付费**。同类型额度按到期时间先后扣减。
- **账单查询与分账**：下载“账单详情”CSV 后，解析 `实例 ID` 字段（格式：`ApiKeyID;业务空间ID;模型名称;输入输出类型;调用渠道;免费额度停用标识`）进行调用溯源。通过资源管理器为 [[workspace]] 绑定标签并启用费用标签，可实现 T+1 级项目/部门分账。完整操作路径见 [账单查询与成本管理](../../raw/model-user-guide/test-1/bill-query-and-cost-management.md)。

## 限制和注意事项
- **额度适用范围**：免费额度仅抵扣实时推理调用，不支持 Batch 调用、上下文缓存、模型训练、部署及 MCP 广场体验。
- **地域策略差异**：免费额度仅对中国内地版生效；全球、国际、美国及欧盟地域模型不提供免费额度，且输入/输出单价因算力调度范围存在区域差异。
- **欠费连锁影响**：账户可用额度 `< 0` 时视为欠费，将直接阻断所有依赖账户余额的服务调用，即使其他模型仍有免费额度亦不可用。建议提前配置预算管理或高额消费预警。
> **注意**：免费额度有效期政策已于 2025年9月8日 调整，新开通用户统一为 90 天，历史用户不受影响，请以控制台展示为准。此外，模型部署时若单次输入超过最长限制或并发超出购买的 TPM，系统将自动降级为按量付费模式，此时推理性能可能下降且受公共限流策略管控。
