# llm application

阿里云百炼平台提供三种 LLM 应用构建范式：智能体（Agent）、工作流（Workflow）与高代码应用，覆盖从零代码配置到深度定制开发的全场景需求。通过原生集成知识库、多模态文件解析、外部工具（MCP/插件）及会话记忆，开发者可快速将大模型能力转化为稳定可控的业务服务或 API。

## 支持的模型与核心功能

* **应用架构选型**
  * `[[agent-application]]`：采用提示词驱动与大模型自主规划架构。新版（2.0）将知识库与外部工具统一为 MCP 协议接入，支持动态调度、多步规划与“思考-执行-反思”全链路可视化。详见 [新版智能体应用](../../raw/application-user-guide/llm-application/new-single-agent-application.md)。
  * `[[workflow-application]]`：基于可视化节点编排，提供大模型、意图分类、变量处理、智能体群组等组件。流程由预定义逻辑严格控制，适合高稳定性要求的固定业务链（如订单处理、数据清洗、报告生成）。
  * `[[rich-code-application]]`：面向开发者提供完整 Python 项目结构，支持 Serverless 函数与 K8s 双模部署。内置一站式 MCP 工具挂载、API 网关、应用可观测性及 Spark Design 前端框架集成，适用于复杂系统对接与私有算法服务。
* **模型兼容**：平台支持千问系列（Max/Plus/Turbo/Long/VL/开源等）、DeepSeek 及自定义部署模型。多模态能力（图片/音视频解析）依赖所选模型原生特性。
* **核心扩展能力**：内置 `[[rag]]` 检索增强、`[[mcp]]` 外部服务调用、多模态文件问答、应用组件复用及短期/长期记忆管理。

## 关键参数

| 参数类别 | 配置项说明 |
|:---|:---|
| **推理控制** | `temperature`（控制生成多样性）、`最长回复长度`。部分模型支持开启 `enable_thinking`（思考模式），提升复杂任务拆解与反思质量。 |
| **上下文与提示词** | 系统提示词支持嵌入自定义变量（如 `/var_name`），用于动态注入业务上下文。可配置角色边界、输出格式与工具调用规则。 |
| **自主决策限制** | `ReAct 最大轮次`（1-50）：限制智能体单次会话内调用工具的最大次数。超限后将强制退出工具链并输出总结。 |
| **文件处理阈值** | **全文引用**：限制“单文件最大解析长度”与“最大拼装长度”（Token），超限从末尾截断。<br>**切片检索**：限制“召回片段数”与“最大拼装长度”，超限按相关性得分倒序剔除。 |
| **记忆与缓存** | 短期记忆支持配置 0-30 轮历史对话。工作流节点支持开启记忆（节点局部缓存或全局自定义缓存）。 |

## 使用方式

1. **控制台配置**：在应用中心创建对应类型应用。智能体通过右侧面板配置模型、Prompt 与技能；工作流通过拖拽节点并绑定输入/输出变量完成链路编排；高代码应用支持上传 `.whl` 包或选择官方模板一键部署。
2. **调试与测试**：配置完成后使用内置对话面板验证功能。支持上传文件进行 `[[rag]]` 检索或全文问答，文件处理逻辑分为三种模式：`全文引用`、`切片检索`（可关联知识库混合检索）、`自定义处理`（交由模型调度外部工具）。详见 [文件问答](../../raw/application-user-guide/llm-application/file-q-a.md)。
3. **发布与 API 集成**：配置必须经过**发布**操作方可生效。发布后获取标准 API Endpoint 与鉴权 Key，支持同步/流式调用。高代码应用可通过配置云原生 API 网关与自定义域名暴露生产级接口，并支持 Token 鉴权与公网访问控制。
4. **版本回滚**：支持查看发布差异记录。在版本管理面板可对历史版本添加描述，或执行**覆盖当前草稿**进行快速回滚。

## 限制与注意事项

> **注意**：旧版智能体（Agent 1.0）与新版（Agent 2.0）底层架构不同，**不支持直接升级或降级**，需重新创建应用迁移配置。此外，早期文档提及“长期记忆 Token 暂不计费”，但新版规划中长期记忆功能处于迭代中，计费策略请以控制台最新公告为准。

* **调用配额**：智能体应用默认限流 **100次/分钟**。该配额为全量 API 共享（含常规对话、文件问答等），高并发场景建议申请配额提升或采用异步处理。
* **文件传输限制**：控制台单次会话限传 10 个文件，单文件 ≤10MB。页面刷新或会话超时会丢失文件。生产环境强烈建议使用**文件上传 API** 获取 `session_file_id` 以支持大文件与稳定传输。
* **工具调用超时**：自定义插件/工具执行存在 **5 秒超时限制**。若第三方接口或本地计算耗时较长，请改用 `[[rich-code-application]]` 实现异步任务队列或调整服务规格。
* **计费模型**：仅按实际调用计费。总成本 = 模型 Token 消耗（输入+输出，含召回文本与记忆内容） + `[[knowledge-base]]` 存储/检索费 + 第三方/特定 MCP 调用费。关闭预解析可节省部分多模态处理开销。
* **合规与备案**：对外提供服务的 AI 应用需完成《生成式人工智能服务管理暂行办法》相关合规备案。部署至生产环境前请确保已获取必要的 RAM 服务关联角色授权。
