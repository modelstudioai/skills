# support

本文档为开发者提供阿里云百炼平台的计费规则、API/SDK 接入指引、关键参数配置及常见技术限制说明，用于快速排障与合规开发。平台功能与协议持续迭代，具体实现细节请以控制台与官方文档为准。详细解答可参考 [常见问题](../../raw/model-user-guide/support/faq-about-alibaba-cloud-model-studio.md) 与合规条款 [相关协议](../../raw/model-user-guide/support/related-agreements.md)。

## 支持的模型/功能
- **模型矩阵**：覆盖千问文本系列（如 `qwen-turbo`、`qwen-max`）及多模态模型（`qwen-vl-plus`），原生支持中、英等 14 种语言及图片微调训练。
- **核心能力**：提供 Completion API 与 Assistant API 推理接口、SFT 模型训练/蒸馏、应用编排及插件/MCP 扩展能力。
- **数据隔离**：支持通过 [[业务空间]] 实现主子账号维度的数据与权限隔离；传输数据全程采用 AES-256 加密，平台承诺不将业务数据用于基座模型训练。

## 关键参数
- **必传字段**：调用推理接口需完整传递 `AppId`、`Prompt`（或对话角色标识）等参数，缺失或格式错误将返回状态码及错误码 `100004`。
- **生成控制**：通过调节 `temperature`、`top_k`、`top_p` 可控制输出随机性；调低参数可有效抑制 [[模型幻觉]]，提升结果确定性，但会降低创造性。`max_tokens` 用于截断生成长度，合理设置可避免冗余输出。
- **计费维度**：模型推理支持后付费（分钟级出账、月结）与预付费（[[节省计划与资源包]]）；训练与部署计费独立核算。具体单价以 [[计费文档]] 实时公示为准。

## 使用方式
- **SDK 接入**：官方维护 Python 与 Java SDK，内置鉴权、重试与流式响应封装。建议优先使用 SDK 替代原生 `curl` 调用，详细依赖安装请参考 [[SDK安装]] 文档。
- **服务管理**：阿里云主账号登录控制台即自动开通服务。因服务开通后暂不支持主动注销，如需停止调用与扣费，请在控制台删除对应的 API-Key。
- **错误定位**：API 返回标准 HTTP 状态码与业务错误码，结合 [[错误排查]] 指南可快速区分网络、鉴权、参数或限流问题。

## 限制和注意事项
- **接口能力边界**：
  - Assistant API 当前暂不支持 `memory` 状态保持，且无法在单次请求中按序自动触发两个本地函数。需拆分为多个请求或创建独立 Assistant 实例。
  - `doc_reference_type` 参数已废弃，新版应用需在控制台界面手动开启“展示答案来源”开关。
  > **注意**：旧版控制台或示例代码中可能仍保留 `doc_reference_type` 配置，该参数在新版应用构建器中已不再生效。迁移旧业务时请以控制台最新交互为准，避免参数无效导致排查耗时。
- **性能与限流策略**：模型生成速度非固定值，受集群负载与您的并发请求数（QPS）影响。触发限流（RPM/RPS）后，重试间隔需根据配额窗口计算（例：120 RPM 限制下，触发限流需等待约 0.8 秒）。
- **合规与版权**：[[万相会员]] 权益与百炼 API 计费体系独立，不可用于抵扣百炼调用费用。产品上架微信小程序等应用商店需提前完成 [[应用合规备案]] 并签署模型合作协议。服务调用受平台 SLA 与三方模型开源协议约束，详细条款请参阅 [相关协议](../../raw/model-user-guide/support/related-agreements.md)。
- **日志与数据留存**：控制台体验页最多展示 100 条历史对话（无时间限制，手动删除后自动递补）；未登录或推理报错的记录不予保存。根据法律法规要求，平台会依法留存模型调用日志与审计数据。
