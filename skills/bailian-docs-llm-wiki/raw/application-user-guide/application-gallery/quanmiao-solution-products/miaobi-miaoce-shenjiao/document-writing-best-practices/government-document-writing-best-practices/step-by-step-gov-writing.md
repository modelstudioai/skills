# 分步式撰写政务稿（精准控制章节内容）

文章主题（prompt）

请以政府的口吻，帮我写一篇通报，篇幅为中篇，标题为：达州市疫情防控最新通报

文章标题

达州市疫情防控最新通报

关键词

疫情、防控、核酸、通告

## 具体操作流程

-   首先选择“创作政务公文”
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750233.png)

-   按系统要求填写【写作内容或主题】，输入【文章标题】、【写作场景】、【关键词】后点【下一步】
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750238.png)

-   查看生成的大纲
    

1).全部换一换

如果对生成的大纲不满意，只需点击“重新生成”即可生成新的大纲。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750236.png)

2).编辑大纲

如果对生成的大纲相对满意，但是需要修改细节，那么只需点击编辑图标进行大纲细节的编辑。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750234.png)

点击引用按钮图标可以添加外部内容或文章链接进行引用。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750237.png)

-   查看生成的摘编
    

如果对生成的摘编不满意，只需点击“重新生成”即可进行优化操作。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750235.png)

-   最后，点击“保存”即可保存此文章至【文档管理】中。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750232.png)

## 素材智搜丰富文章内容

-   点击“搜索素材”，进入到素材搜索界面![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750239.png)
    
-   随后在搜索框内输入您要搜索的内容即可，如：关于搬迁老城区儿童园、动物园的相关文章
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750241.png)

-   您可以将符合您预期的素材内容加入素材库
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750240.png)

-   随后您可以在【素材库】找到您刚刚搜索到并添加的素材内容
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/7781943071/p750242.png)
