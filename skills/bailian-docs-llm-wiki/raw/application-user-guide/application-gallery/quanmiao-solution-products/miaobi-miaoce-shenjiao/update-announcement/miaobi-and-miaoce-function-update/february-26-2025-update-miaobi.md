# 2025年2月26日更新-妙笔

### **迭代说明**

本次迭代妙笔全量化升级，引入Deepseek模型，整体优化了交互界面，写作效果得到提升，写作速度提升，使用门槛进一步降低。

产品地址：[妙笔产品控制台](https://aimiaobi.console.aliyun.com/?product_code=g_broadscope_media&from=bailian#/home)。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921610.png)

### **具体更新细节**

#### **1\. 妙笔接入Deepseek，且写作时支持切换模型**

传媒类-通用文体、法定公文类、事务公文类和营销类写作支持选择模型，新增Deepseek-r1模型可选。

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921590.png)

#### **2\. 每种文体最多支持3种写作方式**

-   直接写作：所有文体都支持，即输入相关信息后可直接生成文章；
    
-   分步骤写作：即按照输入主题->AI写大纲->AI摘编素材->写文章，目前传媒类中的新闻评论和新闻报道支持；![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921661.png)
    
-   高级模式：所有文体都支持高级模式写作，即输入写作主题和指令（prompt）的方式写文章，使写作更加灵活自由。
    
    ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921588.png)
    

#### **3\. 新增写作文体以及重新分类**

1.  文体重新分类，将原先“公文类”和“办公类”文体重新整合分为“法定公文类”和“事务公文类”，并新增9个专业的文体，整体文体以及分类如下：
    
    1.  法定公文：即国务院规定的15种文体，分别是报告、通告、通报、请示、决定、函、决议、公告、公报、命令、批复、议案、意见、纪要、通知；
        
    2.  事务公文：公示、工作汇报、调研报告、意见建议、演讲稿、接待方案、项目书、征求意见、政府意见信、日周月报、学习心得、通用文体。
        
        ![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921593.png)
        

#### **4\. 公文类**文体新增模板输入

能保证对应文体生成的正文都包含关键信息，使输出结果更加专业。![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921604.png)

#### **5\. 支持长文写作**

最多可支持5000字以上的长文的输出，比如演讲稿。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921652.png)

#### **6\. 优化正文输出格式**

-   根据输入的内容模型可生成文章标题；
    
-   优化正文输出格式，将原始的Markdown格式渲染成大小标题。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921594.png)

#### **7\. 优化配图策略**

-   事实类场景，推荐使用“搜索图片”和“来自素材库”；
    
-   创意类场景，推荐使用“AI智能配”。
    

![image.png](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/0599690471/p921591.png)
