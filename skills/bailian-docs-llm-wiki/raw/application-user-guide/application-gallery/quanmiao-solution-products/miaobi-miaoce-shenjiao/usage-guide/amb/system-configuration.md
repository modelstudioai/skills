# 系统配置

## **功能概述**

在系统配置中，用户可以根据需求和偏好进行个性化设置。系统配置包括干预规则和写作信源。

## **功能介绍**

### 干预规则

用户可以通过设置干预规则来调整AI妙笔的行为。干预规则可以用于限制或指导AI在生成内容时的行为，例如关键词干预、意图干预、网页干预。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1021838271/p761987.png)

（1）关键词：支持填入关键词进行干预，一旦触发关键词，即可快速干预。关键词支持的干预范围包括写作提示词、写作结果、搜索词、搜索结果四大类。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1021838271/p761989.png)

（2）意图：支持按照一句话、多句相似的话术进行干预，触发相似话术后，可快速干预。意图话术干预支持写作提示词、搜索词。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1021838271/p761990.png)

（3）网页链接：支持填入文章URL，在搜索召回和内容召回过滤对应文章的URL，不再列为参考文章，不参与搜索生成过程。干预范围支持参考文章和搜索来源。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1021838271/p761991.png)

### 写作信源

写作信源是指用于参考和借鉴的资源。用户可以选择不同的写作信源，如夸克搜索、央视网、新华网、人民网等，以便AI妙笔在生成内容时能够更好地理解和应用相关领域的知识。

![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/1021838271/p761993.png)

**说明**

_注意：信源渠道网站是根据您提供的检索API而定。_

而权重配置会影响AI妙笔写作和搜索结果，权重（采信数量）越高，AI结果会参考的更多。
