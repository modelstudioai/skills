# 妙笔-分布生成创作文章

关于妙笔分布创作文章的相关产品功能说明。

## **功能概述**

分步生成是通过先创建一个基础框架，然后逐步填充详细内容来完成整个文档的过程。分步生成方法有助于系统化创作过程，提高最终产出的质量，同时也可以在每一步中更容易地实施控制和修改。最终也提高了创作效率。[点击前往AI妙笔工作台](https://aimiaobi.console.aliyun.com)

## **具体操作**

文章主题（prompt）

请围绕数字经济释放全新增长动力写一篇新闻

-   选择**分步生成**。分布生成分为四个大步骤，分别是主题、大纲、摘编、文章。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897516.png)
    
-   主题：按照要求填写主题的相关内容，填写完成后点击**下一步**，即可生成大纲。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897529.png)
    
    -   写作内容和主题：填写写作的内容、主题的提示词，需要尽可能详细的描述。
        
    -   写作场景：目前支持四个写作场景，每个场景支持多个文体类型。具体可点击《[首页概览](https://help.aliyun.com/zh/model-studio/amb-homepage-overview)》进行查看。
        
    -   关键词：文章的相关关键词。
        
    -   引用素材：
        
        -   自动补充素材：在写作信源中搜索相关内容进行引用，默认选中。
            
        -   自定义素材：可以添加需要应用到写作中的素材，模型会优先参考。添加的文章模型会优先参考，用于文章创作。
            
            ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897539.png)
            

-   大纲：模型根据主题内容生成大纲内容，用户可以对大纲进行编辑、删除、自定义素材、新增操作，确认无误后点击**下一步**，根据大纲内容可以生成摘编。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897566.png)
    
    -   编辑大纲：如果对生成的大纲相对满意，但是需要修改细节，那么只需点击“编辑”按钮进行大纲细节的编辑。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897577.png)
        
    -   删除大纲：如果对生成的大纲不满意，可以进行删除。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897576.png)
        
    -   新增大纲：用户可以新增大纲的内容。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897579.png)
        
    -   自定义素材：点击**自定义素材**按钮即可搜索相关内容素材进行添加。
        
        **重要**
        
        最多只可添加3条自定义素材。
        
        ![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897586.png)
        
    
-   摘编：模型根据大纲内容可以生成摘编，用户可以对摘编进行编辑、删除操作，确认无误后点击**下一步**，即可开始创作文章。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897588.png)
    
-   文章：模型可以根据主题、大纲、摘编生成文章。![image](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897616.png)
    
    -   采纳：当用户对生成结果满意，即可点击**采纳**。
        
    -   弃用：当用户对生成结果不满意，即可点击**弃用**。
        
    -   换一换：支持对当前生成的文本进行**直接换**、**缩写**、**扩写**。
        
    -   结果干预：可以更改素材源，进行重新生成文本。
        

当用户满意当前生成的内容，点击**采纳**，即可生成对应的文章。![3539751dbc18d6ddf8b2dd3951618d40](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897618.png)

## **快捷操作**

鼠标选中需要修改优化的内容，即会弹出快捷操作悬浮窗口，可以对选中内容进行一系列操作，包含：生成标题、生成摘要、全文续写、全文改写、全文翻译、抽取关键字、一键润色、文章审核、用户观点预测。![c66eb12b5047e17b0c1e8cccadd0be11](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/9634555371/p897621.png)

## **AI工具箱**

AI工具箱支持对当前文本进行一些智能操作，包含：生成标题、生成摘要、全文续写、全文改写、全文翻译、抽取关键字、一键润色、文章审核等。
